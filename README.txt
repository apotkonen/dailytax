Font issues

Fonts are changed to free versions to avoid pdf-preview blockage. Assumed
blockage cause is version controlled software unneeded versions removal-
shrinkage operation caused pdf reinitiation to system, and possible some
previewer font related costs are generated for each introduced "new pdf" which
then led to account preview blockage. Now changed to fonts not adding any extra
cost. Same has to be done for other repositories pdf's under this account.
